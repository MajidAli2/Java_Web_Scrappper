# Java_Web_Scrapepper

A Java-based web scraper application for extracting and processing data from websites.

## Features
- Web scraping capabilities
- Data extraction and processing
- Built with Java
- Easy-to-use command-line interface

## Project Structure
```
Web_Scraper/
├── src/
│   └── WebScraperApp.java
├── lib/
├── run.bat
├── run.ps1
├── README.md
└── LICENSE
```

## Installation

1. Clone the repository:
```bash
git clone https://github.com/MajidAli2/Java_Web_Scrappper.git
cd Java_Web_Scrappper
```

2. Compile the Java files:
```bash
javac src/WebScraperApp.java
```

## Usage

Run the scraper using the provided scripts:

**On Windows (PowerShell):**
```bash
.\run.ps1
```

**On Windows (Command Prompt):**
```bash
run.bat
```

Or run directly:
```bash
java -cp src WebScraperApp
```

## Requirements
- Java 8 or higher
- Internet connection for web scraping
